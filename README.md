# Java Input Output Program

## 📌 Description
This is a simple Java program that takes two integer inputs from the user and calculates their sum.

## 🚀 Features
- Takes user input using Scanner
- Performs addition of two numbers
- Handles invalid input using exception handling

## 🛠️ Technologies Used
- Java
- Scanner Class

## ▶️ How to Run

1. Compile the program:
   
javac InputOutput.java


3. Run the program:

java InputOutput


## 📷 Example Output

Enter the value of a: 5
Enter the value of b: 10
The sum of the numbers is: 15
